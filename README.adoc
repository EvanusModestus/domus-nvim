= domus-nvim
:description: Hierarchical, modular Neovim configuration
:icons: font

A modern, hierarchical Neovim configuration with embedded documentation.

== Features

* **Hierarchical Structure** - Organized by domain (config, core, plugins, features, ui)
* **Separated Concerns** - Plugin specs vs configs
* **Native ftplugin** - Filetype settings via Neovim's native mechanism
* **Embedded Antora Docs** - Documentation as code

=== Installing Neovim

Ubuntu/Debian (PPA for latest):

[source,bash]
----
sudo add-apt-repository ppa:neovim-ppa/unstable
sudo apt update && sudo apt install neovim
----

Arch Linux:

[source,bash]
----
sudo pacman -S neovim
----

macOS:

[source,bash]
----
brew install neovim
----

Build from source: https://github.com/neovim/neovim/blob/master/BUILD.md

== Installation

=== Back up existing config

If you have an existing Neovim configuration, move it out of the way first:

[source,bash]
----
mv ~/.config/nvim ~/.config/nvim.bak
mv ~/.local/share/nvim ~/.local/share/nvim.bak
mv ~/.local/state/nvim ~/.local/state/nvim.bak
mv ~/.cache/nvim ~/.cache/nvim.bak
----

=== Clone and launch

[source,bash]
----
git clone https://github.com/EvanusModestus/domus-nvim ~/.config/nvim
nvim
----

Or run alongside an existing config using `NVIM_APPNAME`:

[source,bash]
----
git clone https://github.com/EvanusModestus/domus-nvim ~/.config/nvim-domus
NVIM_APPNAME=nvim-domus nvim

# Create an alias for convenience
alias v='NVIM_APPNAME=nvim-domus nvim'
----

=== First launch

On the first run:

1. **lazy.nvim** bootstraps itself automatically (cloned from GitHub)
2. **~90 plugins** are downloaded and installed — this takes a minute or two
3. **Mason** installs LSP servers, formatters, and linters in the background
4. **TreeSitter** parsers compile on first encounter of each filetype

Let the initial install complete before editing files. You can check Mason progress with `:Mason` and lazy.nvim status with `:Lazy`.

=== Uninstall

Remove the configuration and all associated data:

[source,bash]
----
rm -rf ~/.config/nvim
rm -rf ~/.local/share/nvim
rm -rf ~/.local/state/nvim
rm -rf ~/.cache/nvim
----

If installed with `NVIM_APPNAME=nvim-domus`:

[source,bash]
----
rm -rf ~/.config/nvim-domus
rm -rf ~/.local/share/nvim-domus
rm -rf ~/.local/state/nvim-domus
rm -rf ~/.cache/nvim-domus
----

----

│   │   ├── lazy.lua
│   │   └── util.lua
│   ├── plugins/                # Plugin system
│   │   ├── specs/              # What to load
│   │   │   ├── editor.lua
│   │   │   ├── ui.lua
│   │   │   ├── coding.lua

== Key Bindings

Leader: `<Space>`

=== Navigation

* `<Space>ff` - Find files
* `<Space>fg` - Live grep
* `<Space>e` - File browser (Oil)
* `<C-e>` - Harpoon menu

=== LSP

* `gd` - Go to definition
* `gr` - References
* `K` - Hover docs
* `<Space>ca` - Code action

=== Git

* `<Space>gs` - Git status
* `<Space>lg` - LazyGit
* `<Space>hs` - Stage hunk
