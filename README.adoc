= domus-nvim
:description: Hierarchical, modular Neovim configuration
:icons: font

A modern, hierarchical Neovim configuration with embedded documentation.

== Features

* **Hierarchical Structure** - Organized by domain (config, core, plugins, features, ui)
* **Separated Concerns** - Plugin specs vs configs
* **Native ftplugin** - Filetype settings via Neovim's native mechanism
* **Embedded Antora Docs** - Documentation as code

=== Installing prerequisites

==== Arch Linux

[source,bash]
----
sudo pacman -S neovim git base-devel ripgrep fd lazygit tmux npm python ipython
----

Install a Nerd Font (e.g., JetBrainsMono):

[source,bash]
----
sudo pacman -S ttf-jetbrains-mono-nerd
----

==== Ubuntu/Debian

[source,bash]
----
# Neovim (PPA for 0.11+)
sudo add-apt-repository ppa:neovim-ppa/unstable
sudo apt update
sudo apt install neovim git build-essential ripgrep fd-find nodejs npm python3 python3-pip

# fd is installed as fdfind on Debian/Ubuntu — alias it
sudo ln -sf $(which fdfind) /usr/local/bin/fd
----

lazygit (not in default repos):

[source,bash]
----
LAZYGIT_VERSION=$(curl -s "https://api.github.com/repos/jesseduffield/lazygit/releases/latest" | grep -Po '"tag_name": "v\K[^"]*')
curl -Lo lazygit.tar.gz "https://github.com/jesseduffield/lazygit/releases/latest/download/lazygit_${LAZYGIT_VERSION}_Linux_x86_64.tar.gz"
tar xf lazygit.tar.gz lazygit
sudo install lazygit /usr/local/bin
----

Nerd Font — download from https://www.nerdfonts.com/font-downloads[nerdfonts.com] and install to `~/.local/share/fonts/`, then run `fc-cache -fv`.

==== macOS

[source,bash]
----
brew install neovim git ripgrep fd lazygit tmux node python ipython
brew install --cask font-jetbrains-mono-nerd-font
----

==== Language toolchains

Mason auto-installs most LSP servers and formatters, but some languages require their own toolchain installed first.

**Rust** — rust-analyzer, rustfmt, and clippy all come from rustup:

[source,bash]
----
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
rustup component add rust-analyzer clippy
----

**Python** — pyright is installed by Mason, but you need the runtime and optionally ipython for the REPL (`<Space>sc` to send code):

[source,bash]
----
pip install ipython
----

**Node.js** — required by several Mason-installed servers (ts_ls, html, cssls, jsonls, yamlls). Install via your package manager (shown above) or https://github.com/nvm-sh/nvm[nvm].

Build Neovim from source: https://github.com/neovim/neovim/blob/master/BUILD.md

== Installation

=== 1. Back Up Existing Config

If you already have a Neovim configuration:

[source,bash]
----
mv ~/.config/nvim ~/.config/nvim.bak
mv ~/.local/share/nvim ~/.local/share/nvim.bak
mv ~/.local/state/nvim ~/.local/state/nvim.bak
mv ~/.cache/nvim ~/.cache/nvim.bak
----

=== 2. Clone and Launch

[source,bash]
----
git clone https://github.com/EvanusModestus/domus-nvim.git ~/.config/nvim
nvim
----

On first launch:

1. **lazy.nvim** bootstraps itself (cloned from GitHub)
2. **~90 plugins** download and install — takes 1-2 minutes
3. **Mason** installs LSP servers, formatters, and linters in the background
4. **TreeSitter** parsers compile on first encounter of each filetype

Let the initial install complete before editing files.

=== 3. Verify

[source,bash]
----
for cmd in nvim git gcc rg fd lazygit tmux node python3; do
  printf "%-10s %s\n" "$cmd" "$(command -v $cmd >/dev/null 2>&1 && echo 'OK' || echo 'MISSING')"
done
----

Inside nvim:

* `:Lazy` — all plugins loaded?
* `:Mason` — LSP servers installed?
* `:checkhealth` — any errors?

=== Troubleshooting

[cols="1,2"]
|===
| Problem | Fix

| Plugins fail to clone
| Check SSH key: `ssh -T git@github.com`. If no key, lazy.nvim uses HTTPS automatically.

| Mason installs fail
| `:Mason` → check for errors. Most need `npm` or `python3` installed.

| No icons / broken characters
| Nerd Font not installed or terminal not using it. Set terminal font to JetBrainsMono Nerd Font.

| TreeSitter parser errors
| Need `gcc` or `clang`: `sudo pacman -S base-devel` (Arch) or `sudo apt install build-essential` (Ubuntu).

| "E5108: Error executing lua"
| Neovim too old. Need 0.11.0+: `nvim --version`.
|===

=== Uninstall

[source,bash]
----
rm -rf ~/.config/nvim
rm -rf ~/.local/share/nvim
rm -rf ~/.local/state/nvim
rm -rf ~/.cache/nvim
----

== Key Bindings

Leader: `<Space>`

=== Navigation

* `<Space>ff` - Find files
* `<Space>fg` - Live grep
* `<Space>e` - File browser (Oil)
* `<C-e>` - Harpoon menu

=== LSP

* `gd` - Go to definition
* `gr` - References
* `K` - Hover docs
* `<Space>ca` - Code action

=== Git

* `<Space>gs` - Git status
* `<Space>lg` - LazyGit
* `<Space>hs` - Stage hunk
