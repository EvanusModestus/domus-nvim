= Keymaps Reference
:description: Complete keybindings reference for domus-nvim — navigation, editing, LSP, git, debug, and filetype-specific bindings
:navtitle: Keymaps
:icons: font

Leader key: `{leader}`

All keymaps are defined in `{lua-dir}/config/keymaps.lua` (global) and individual plugin configs (plugin-specific).
Press `{leader}` and wait for which-key to display available continuations.

== Text Objects

All objects work with standard Vim operators: `y` (yank), `d` (delete), `c` (change), `v` (visual select).

=== AsciiDoc

[cols="1,2", options="header"]
|===
| Object | Target

| `i-` / `a-`
| Inside / around `----` (code/listing block)

| `i/` / `a/`
| Inside / around `////` (comment block)

| `i=` / `a=`
| Inside / around `====` (example/sidebar block)

| `i_`
| Inside `__` (italic inline)

| `i*`
| Inside `**` (bold inline)

| `i+`
| Inside `++` (passthrough inline)
|===
