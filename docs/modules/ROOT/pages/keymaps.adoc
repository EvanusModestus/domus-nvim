= Keymaps Reference
:description: Complete keybindings reference

Leader key: `{leader}`

== Navigation

[cols="1,2,1"]
|===
| Key | Action | Plugin

| `{leader}ff`
| Find files
| Telescope

| `{leader}fg`
| Live grep
| Telescope

| `{leader}fb`
| Buffers
| Telescope

| `<C-p>`
| Git files
| Telescope

| `{leader}e` or `-`
| File browser
| Oil

| `<C-e>`
| Harpoon menu
| Harpoon

| `{leader}1-4`
| Harpoon files 1-4
| Harpoon
|===

== Editing

[cols="1,2"]
|===
| Key | Action

| `J` (visual)
| Move selection down

| `K` (visual)
| Move selection up

| `{leader}y`
| Yank to system clipboard

| `{leader}p`
| Paste without yanking

| `{leader}d`
| Delete to void register

| `jk` or `jj`
| Exit insert mode
|===
