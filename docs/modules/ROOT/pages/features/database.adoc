= Database Integration
:description: SQL database client, visual browser, and schema-aware autocompletion inside Neovim via vim-dadbod, vim-dadbod-ui, and vim-dadbod-completion.
:navtitle: Database
:icons: font

{database} provides a database client inside Neovim, combined with vim-dadbod-ui for a visual interface and vim-dadbod-completion for SQL autocompletion through {completion-engine}.

== Components

=== {database}

Core database interaction engine.
Supports PostgreSQL, MySQL, SQLite, SQL Server, MongoDB, Redis, and more.

Run queries directly with the `:DB` command:

[source,vim]
----
:DB postgresql://localhost/mydb SELECT * FROM users;
----

The connection string follows standard URI format.
Results appear in a scratch buffer for inspection.

=== vim-dadbod-ui

Visual database browser providing a persistent sidebar for database exploration.

[cols="1,3", options="header"]
|===
| Command | Action

| `:DBUI`
| Open the database browser sidebar

| `:DBUIToggle`
| Toggle the browser panel open or closed
|===

Features:

* Browse schemas, tables, and views in a tree structure
* Execute queries from any buffer connected to a database
* Save queries for reuse across sessions
* Multiple database connections managed simultaneously

=== vim-dadbod-completion

SQL keyword and schema-aware completion source.
Integrates with {completion-engine} for automatic suggestions while writing queries -- table names, column names, SQL keywords, and functions appear in the completion menu without manual invocation.
