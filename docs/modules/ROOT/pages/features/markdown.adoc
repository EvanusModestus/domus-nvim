= Markdown Support
:description: Markdown editing with live browser preview, auto-lists, table alignment, enhanced syntax highlighting, and filetype-specific settings.
:navtitle: Markdown
:icons: font

Comprehensive Markdown support with live preview, enhanced editing, automatic list management, and table alignment.

== Plugins

=== {markdown-preview}

Browser-based live preview that auto-syncs scroll position with the editor cursor.
Includes WSL browser detection for seamless cross-environment usage.

`:MarkdownPreview`:: Open live preview in the default browser.
`:MarkdownPreviewStop`:: Close the preview window.

=== vim-markdown (preservim)

Enhanced Markdown editing with improved syntax highlighting, concealment, and header navigation.
Folding is disabled -- treesitter handles structural folding instead.

Provides motions for navigating between headers and following links within the document.

=== bullets.vim

Automatic list continuation and management.
Enabled in markdown, text, and gitcommit filetypes.

* Auto-continue lists on `Enter` -- pressing Enter after a list item creates the next item
* Checkbox toggling between `- [ ]` and `- [x]`
* Nested list indentation with `>` and `<`
* Works with `-`, `*`, and numbered list styles

=== vim-table-mode

Auto-align table columns as you type.

`:TableModeToggle`:: Enable or disable table mode for the current buffer.
`{leader}tm`:: Toggle table mode via keymap.

Formats existing tables and keeps column alignment consistent while editing.

== Filetype Settings

Defined in `{ftplugin-dir}/markdown.lua`:

[cols="1,3", options="header"]
|===
| Setting | Behavior

| Spell checking
| Enabled with `en_us` dictionary

| Text wrapping
| Soft wrap with `linebreak` (no hard line breaks inserted)

| Auto-comment
| Comment continuation disabled (new lines after comments are plain text)

| `gf` mapping
| Remapped to follow Markdown links in `[text](path.md)` format
|===

== Configuration

Plugin declarations live in `{lua-dir}/plugins/specs/lang/markdown.lua`.
Filetype-specific settings are in `{ftplugin-dir}/markdown.lua`.
