= Git Integration
:description: Git workflow tools in domus-nvim including Fugitive, Gitsigns, LazyGit, Diffview, and Telescope git pickers.
:navtitle: Git
:icons: font

== Overview

domus-nvim integrates five git tools, each serving a distinct role in the workflow:

* **vim-fugitive** -- full git command interface within Neovim
* **gitsigns.nvim** -- buffer-level git decorations and hunk operations
* **LazyGit** -- full-featured terminal UI for git (via lazygit.nvim)
* **diffview.nvim** -- professional multi-file diff and merge viewing
* **{fuzzy-finder}** -- git commit and branch pickers

== Fugitive

vim-fugitive exposes the full `git` command set as `:Git` ex-commands, with deep buffer integration for staging, diffing, and conflict resolution.

[cols="1,3", options="header"]
|===
| Key / Command | Action

| `{leader}gs`
| Git status (opens `:Git` summary buffer)

| `{leader}gp`
| Git push

| `:Git blame`
| Inline blame annotations for every line

| `:Gvdiffsplit`
| Vertical diff split against the index
|===

== Gitsigns

gitsigns.nvim provides per-line git status in the sign column and a complete set of hunk operations without leaving the buffer.

=== Signs

[cols="1,2", options="header"]
|===
| Sign | Meaning

| `+`
| Added lines

| `~`
| Changed lines

| `_`
| Deleted lines
|===

=== Hunk Navigation

[cols="1,3", options="header"]
|===
| Key | Action

| `]c`
| Next hunk

| `[c`
| Previous hunk
|===

=== Hunk Operations

[cols="1,3", options="header"]
|===
| Key | Action

| `{leader}hs`
| Stage hunk

| `{leader}hr`
| Reset hunk

| `{leader}hS`
| Stage entire buffer

| `{leader}hu`
| Undo stage hunk

| `{leader}hR`
| Reset entire buffer

| `{leader}hp`
| Preview hunk inline

| `{leader}hb`
| Blame current line

| `{leader}hd`
| Diff this file against the index
|===

== LazyGit

LazyGit provides a full terminal UI for git operations -- staging, committing, branching, rebasing, stashing, and cherry-picking -- all through a keyboard-driven interface.

`{leader}lg` -- Open LazyGit in a floating terminal.

== Diffview

diffview.nvim provides a professional multi-file diff interface with a file panel, inline diff rendering, and merge conflict resolution tools.

[cols="1,3", options="header"]
|===
| Key | Action

| `{leader}gd`
| Open Diffview (diff working tree against index)

| `{leader}gh`
| File history (git log for the current file)
|===

== Telescope Git Pickers

{fuzzy-finder} provides fuzzy-searchable git pickers for commits and branches:

[cols="1,3", options="header"]
|===
| Key | Action

| `{leader}gc`
| Browse commits with diff preview

| `{leader}gb`
| Browse and switch branches
|===

== Configuration

[cols="1,2", options="header"]
|===
| File | Purpose

| `{lua-dir}/plugins/config/git.lua`
| Fugitive keymaps, gitsigns setup, and hunk operations

| `{lua-dir}/plugins/config/lazygit.lua`
| LazyGit floating terminal configuration

| `{lua-dir}/plugins/specs/git.lua`
| Plugin declarations for all git plugins
|===
