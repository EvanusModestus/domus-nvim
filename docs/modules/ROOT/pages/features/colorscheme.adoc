= Colorscheme
:description: Catppuccin Mocha with Tokyo Night-inspired palette overrides, 50+ custom highlight groups, and 5 alternative themes.
:navtitle: Colorscheme
:icons: font

Catppuccin Mocha serves as the primary theme with deep customization -- refined base colors inspired by Tokyo Night's blue palette, 50+ custom highlight groups, and tight integration with every major plugin.

== Custom Palette

The default Catppuccin Mocha palette is overridden with Tokyo Night-inspired blues for a cooler, more refined aesthetic.

== Custom Highlight Groups

Over 50 highlight groups are customized across these categories:

Treesitter nodes:: Keywords, functions, strings, types, operators, and comments with distinct color assignments for visual parsing.

Markup:: Bold, italic, inline code, links, and heading levels 1--6 each have unique styling.

Diff:: Added, removed, and changed lines use green, red, and blue respectively for clear Git integration.

Rainbow delimiters:: Nested brackets and parentheses cycle through distinct colors for depth tracking.

LSP diagnostics:: Error, warning, info, and hint each have foreground, underline, and virtual text styles.

UI elements:: Cursor line, line numbers, sign column, floating windows, and popup menus.

Plugin integrations:: Highlight overrides for plugin-specific highlight groups ensuring visual consistency.

== Customizing Highlights

All highlight overrides live in the `custom_highlights` function:

[source,lua]
----
custom_highlights = function(colors)
    return {
        Visual = {
            bg = "#454025",  -- muted yellow tint
            bold = true,
        },
        -- add overrides here
    }
end
----

Use `colors.<name>` for palette variables or hex strings (`#RRGGBB`) for precise control.
Increase hex values to brighten, decrease to darken.

After editing, reload the colorscheme with `:source %` or restart Neovim.
