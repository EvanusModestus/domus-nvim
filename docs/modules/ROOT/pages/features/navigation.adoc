= Navigation
:description: File and code navigation in domus-nvim with Telescope fuzzy finding, Harpoon quick marks, Oil file browsing, and Flash labeled jumps.
:navtitle: Navigation
:icons: font

== Overview

domus-nvim provides four navigation systems, each optimized for a different access pattern:

* **{fuzzy-finder}** -- fuzzy finding across files, grep, buffers, and more
* **Harpoon** -- persistent quick marks for frequently accessed files
* **{file-browser}** -- file system browsing as an editable buffer
* **Flash** -- labeled character jumps for rapid in-buffer movement

== Telescope

{fuzzy-finder} is the primary search interface, extended with fzf-native for compiled C fuzzy matching performance.

=== Ripgrep Configuration

Telescope's live grep uses these ripgrep defaults:

* `--no-config` -- ignores user ripgrep config to ensure consistent behavior
* `--smart-case` -- case-insensitive unless the query contains uppercase
* `--hidden` -- includes hidden files (dotfiles)
* `--glob=!.git/` -- excludes the `.git` directory

=== Layout

Telescope uses a flex layout that adapts between horizontal and vertical orientation based on window dimensions.

== Harpoon

Harpoon provides persistent quick marks -- a short list of files you return to constantly during a session.
Unlike buffers or recent files, Harpoon marks survive across Neovim restarts and are project-scoped.

[cols="1,3", options="header"]
|===
| Key | Action

| `{leader}a`
| Add current file to Harpoon list

| `<C-e>`
| Open Harpoon menu (reorder, remove)

| `{leader}1`
| Navigate to Harpoon file 1

| `{leader}2`
| Navigate to Harpoon file 2

| `{leader}3`
| Navigate to Harpoon file 3

| `{leader}4`
| Navigate to Harpoon file 4
|===

== Oil

{file-browser} presents the file system as an editable Neovim buffer.
Renaming, moving, and deleting files is done through normal buffer editing operations -- change text, then save the buffer to apply the file system mutations.

Oil buffers are excluded from {session-manager} to prevent stale directory listings from being restored.

[cols="1,3", options="header"]
|===
| Key | Action

| `-` or `{leader}e`
| Open Oil in the current file's directory

| `<CR>`
| Open file or enter directory

| `-`
| Navigate to parent directory

| `g.`
| Toggle hidden files
|===

== Flash

Flash provides labeled character jumps -- type a search character, and Flash labels every match on screen with a unique key.
Press the label key to jump directly, eliminating the need for repeated `f`/`t` or `w` presses.

[cols="1,3", options="header"]
|===
| Key | Action

| `s`
| Flash jump (normal mode)

| `S`
| Flash treesitter select (select a treesitter node by label)

| `r`
| Remote operator (operator-pending mode -- apply an operator to a labeled location without moving the cursor)
|===
