== Requirements

[cols="1,1,2", options="header"]
|===
| Component | Version | Notes

| Neovim
| {neovim-version}
| Required — `vim.lsp.config` API (mason-lspconfig)

| Git
| 2.19+
| Plugin installation via {plugin-manager}

| C compiler
| gcc or clang
| Required — telescope-fzf-native and LuaSnip jsregexp build natively

| make
| Any
| Required — same native build steps

| Nerd Font
| Any https://www.nerdfonts.com[Nerd Font]
| Required — icons in statusline, file browser, dashboard, which-key

| Node.js
| 18+
| Optional — for LSP servers (pyright, ts_ls, jsonls, yamlls, etc.)

| ripgrep
| Any
| Recommended — Telescope live grep

| fd
| Any
| Recommended — faster Telescope file finding

| lazygit
| Any
| Optional — `{leader}lg` integration

| tmux
| Any
| Optional — `Ctrl-f` tmux-sessionizer binding
|===

NOTE: {completion-engine} ships prebuilt binaries — no Rust toolchain needed for completion.

== Platform Packages

=== Arch Linux

[source,bash]
----
sudo pacman -S neovim git base-devel ripgrep fd lazygit tmux npm python ipython
----

Nerd Font:

[source,bash]
----
sudo pacman -S ttf-jetbrains-mono-nerd
----

=== Ubuntu / Debian

[source,bash]
----
# Neovim PPA (0.11+)
sudo add-apt-repository ppa:neovim-ppa/unstable
sudo apt update
----

[source,bash]
----
# Core packages
sudo apt install -y neovim git build-essential ripgrep fd-find nodejs npm python3 python3-pip
----

[source,bash]
----
# fd is installed as fdfind on Ubuntu — symlink it
sudo ln -sf $(which fdfind) /usr/local/bin/fd
----

[source,bash]
----
# lazygit (not in default repos)
LAZYGIT_VERSION=$(curl -s "https://api.github.com/repos/jesseduffield/lazygit/releases/latest" \
  | grep -Po '"tag_name": "v\K[^"]*')
curl -Lo /tmp/lazygit.tar.gz \
  "https://github.com/jesseduffield/lazygit/releases/latest/download/lazygit_${LAZYGIT_VERSION}_Linux_x86_64.tar.gz"
tar xf /tmp/lazygit.tar.gz -C /tmp lazygit
sudo install /tmp/lazygit /usr/local/bin
----

Nerd Font (CLI install):

[source,bash]
----
mkdir -p ~/.local/share/fonts
curl -fLo /tmp/JetBrainsMono.tar.xz \
  https://github.com/ryanoasis/nerd-fonts/releases/latest/download/JetBrainsMono.tar.xz
tar xf /tmp/JetBrainsMono.tar.xz -C ~/.local/share/fonts/
fc-cache -fv
----

Set your terminal emulator's font to "JetBrainsMono Nerd Font".

=== macOS

[source,bash]
----
brew install neovim git ripgrep fd lazygit tmux node python ipython
brew install --cask font-jetbrains-mono-nerd-font
----

== Language Toolchains

{lsp-installer} auto-installs most LSP servers and formatters.
Some languages require their own toolchain installed first.

Rust:: rust-analyzer, rustfmt, and clippy come from rustup:
+
[source,bash]
----
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
rustup component add rust-analyzer clippy
----

Python:: pyright is installed by {lsp-installer}, but you need the runtime and optionally ipython for the REPL (`{leader}sc` to send code):
+
[source,bash]
----
# Arch
pip install ipython

# Ubuntu/Debian (pip is blocked — use apt)
sudo apt install python3-ipython
----

Node.js:: Required by several Mason-installed servers (ts_ls, html, cssls, jsonls, yamlls).
Install via your package manager (shown above) or https://github.com/nvm-sh/nvm[nvm].

== Installation

=== Standard Install

[source,bash,subs=attributes+]
----
# Back up existing config
mv {config-dir} {config-backup}
mv ~/.local/share/nvim ~/.local/share/nvim.bak
mv ~/.local/state/nvim ~/.local/state/nvim.bak
mv ~/.cache/nvim ~/.cache/nvim.bak

# Clone
git clone {repo-url} {config-dir}

# Launch with NVIM_APPNAME
NVIM_APPNAME={appname} nvim
----

=== Side-by-Side Install (NVIM_APPNAME)

Run alongside an existing config without conflict:

[source,bash,subs=attributes+]
----
git clone {repo-url} {appname-dir}

# Run with NVIM_APPNAME
NVIM_APPNAME={appname} nvim

# Or create an alias
alias v='NVIM_APPNAME={appname} nvim'
----

`NVIM_APPNAME` isolates data directories automatically:

[cols="1,2", options="header"]
|===
| Directory | Path

| Config
| `{appname-dir}/`

| Data
| `~/.local/share/{appname}/`

| State
| `~/.local/state/{appname}/`

| Cache
| `~/.cache/{appname}/`
|===

=== Development Install (Symlink)

For active development on the config itself:

[source,bash,subs=attributes+]
----
# Clone to your projects directory
git clone {repo-url} ~/projects/{repo-name}

# Symlink into config
ln -s ~/projects/{repo-name} {appname-dir}

# Launch
NVIM_APPNAME={appname} nvim
----

== Post-Install

On first launch:

1. **{plugin-manager}** bootstraps itself (cloned from GitHub)
2. **~90 plugins** download and install — takes 1-2 minutes
3. **{lsp-installer}** installs LSP servers, formatters, and linters in the background
4. **{treesitter}** parsers compile on first encounter of each filetype

Let the initial install complete before editing files.

== Verify

[source,bash]
----
for cmd in nvim git gcc rg fd lazygit tmux node python3; do
  printf "%-10s %s\n" "$cmd" \
    "$(command -v $cmd >/dev/null 2>&1 && echo 'OK' || echo 'MISSING')"
done
----

Inside Neovim:

* `:Lazy` — all plugins loaded?
* `:Mason` — LSP servers installed?
* `:checkhealth` — any errors?

== Troubleshooting

[cols="1,2", options="header"]
|===
| Problem | Fix

| Plugins fail to clone
| Check SSH key: `ssh -T git@github.com`.
domus-nvim uses HTTPS URLs by default, so SSH is not required.

| Mason installs fail
| `:Mason` -> check for errors.
Most tools need `npm` or `python3` installed.

| No icons / broken characters
| Nerd Font not installed or terminal not using it.
Set terminal font to "JetBrainsMono Nerd Font".

| TreeSitter parser errors
| Need `gcc` or `clang`: `sudo pacman -S base-devel` (Arch) or `sudo apt install build-essential` (Ubuntu).

| "E5108: Error executing lua"
| Neovim too old.
Need {neovim-version}: check with `nvim --version`.

| Clipboard not working on WSL
| Install `win32yank.exe`: `choco install win32yank` or download from GitHub.
domus-nvim auto-detects WSL and configures the provider.
|===

== Uninstall

[source,bash,subs=attributes+]
----
rm -rf {appname-dir}
rm -rf ~/.local/share/{appname}
rm -rf ~/.local/state/{appname}
rm -rf ~/.cache/{appname}
----
